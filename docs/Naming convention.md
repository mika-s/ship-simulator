# Naming convention

## Filenames

### Component filenames

Should start with uppercase letter.

### Business logic filenames

Should start with lowercase letter.

Extensions:

| Variable       | Extension    |
|----------------|--------------|
| General files  | *.js         |
| Tests          | *.test.js    |
| Utility        | *.util.js    |
| Reducers       | *.reducer.js |
